Method

Method: every agent is the same iterative-deepening alpha-beta solver (flip-solver.js); a rung is a per-move time budget, and "random" plays uniformly random legal moves. Deals are seeded and reproducible. Add data with node analyze.js ladder --json analysis/data.json or node analyze.js selfplay --json analysis/data.json, then rebuild this page with node analyze.js report.
